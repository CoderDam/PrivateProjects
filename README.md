# PrivateProjects

## Pop Groups
Big project started in july 2017, developped with:  
  - **React** (+ Redux)  
  - **Node.js** (+ Express)  
  - **websocket** (Socket.io)  
  - **MySQL** (+ Sequelize.js)  

_Concept_: management game where players can recruit young people (_idols_ like in Japanese or Korean pop) in order to build music groups.
